sudo

Description:â€‹

Sudo is a command line program that allows users to be given the ability to run certain (or all) commands as root.

Installationâ€‹

Sudo is included into every major linux distro. If you need to update follow these steps:

sudo apt-get update
sudo apt-get upgrade

Configuration:â€‹

Add User to Sudo Fileâ€‹

sudo /usr/sbin/sudoers

Example: Grant User Permission to Run All Commands as with sudo (make sure you 100% trust this user)â€‹

insert_username ALL=(ALL) ALL

Configure User to Run Certain Commands with sudoâ€‹

User_name Machine_name=(Effective_user) command

user_name = the name of the sudo user
machine_name = host name of machine
effective_user = users that are allowed to execute said commands
command = command users are allowed to run with sudo

Allow user nathan to execute any command with sudo on host ubuntuâ€‹

nathan ubuntu=(ALL) ALL

Allow user nathan to execute ssh with sudo on host ubuntuâ€‹

nathan ubuntu=(nathan) ssh

Commands:â€‹

List Sudo Permissionsâ€‹

sudo -l

Reset sudo timestamp (ticket).â€‹

This forces the user to input sudo password next time they run the commandâ€‹

sudo -K

Run a command as another user or group (can ommit group)â€‹

sudo -u user -g group id -a

Launch default shell with sudo permissionsâ€‹

sudo -i
